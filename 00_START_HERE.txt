⚠️  IMPORTANT RULES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ DO:
   • Keep GitHub repository PUBLIC (Netlify needs access)
   • Include MEDICAL DISCLAIMER on every article
   • Link to authoritative sources (IAP, WHO, CDC)
   • Test locally BEFORE pushing to GitHub
   • Use simple, parent-friendly language
   • Update content regularly

❌ DON'T:
   • Make the repo PRIVATE
   • Skip medical disclaimers
   • Invent medical facts
   • Diagnose or prescribe
   • Push without testing
   • Use medical jargon


📝 ADDING CONTENT IS SIMPLE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TO ADD NEW ARTICLE:
  1. Copy:    education/template-article.html
  2. Rename:  education/my-article.html
  3. Edit:    Replace [BRACKETED SECTIONS]
  4. Update:  Add link in education/index.html
  5. Test:    python -m http.server 8000
  6. Push:    git add . && git commit -m "Add article" && git push
  7. Live:    Within 1-2 minutes!

TO ADD SYMPTOM GUIDE:
  1. Copy:    symptoms/template-symptom-guide.html
  2. Rename:  symptoms/fever.html
  3. Edit:    Add symptoms, red flags, assessment logic
  4. Test & Push: Same as above

Same process for nutrition guides, tools, vaccination schedules!
